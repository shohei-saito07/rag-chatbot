# rag-chatbot
## 社内ドキュメント検索RAGチャットボット

OllamaとStreamlitを使ってWebアプリを作成しました。  
APIキー不要・完全無料のRAGチャットボットです。  
社内規定やマニュアルを自然言語で検索できるようになっています。

## 作成した背景・目的

SESエンジニアとして働いていく中でAI技術に興味を持ち、  
自社内で使用できないかの考えて試しで作成してみました。

「まずは動作するもの」という目標に
業務でよく使う社内ドキュメント検索をAIで解決できないかと考えてこのアプリを作成しました。

## 機能一覧

- 自然言語で社内規定を検索できる
- OllamaのローカルLLMで回答を生成する
- ドキュメントをUI上から追加できる
- 回答の参照元ドキュメントを表示する

## 使用技術

- Python 3.11
- Streamlit（WebUI)
- Ollama / llama3.2(ローカルLLM)
- NumPy（ベクトル検索）

## セットアップ手順

1. Ollamaをインストール
2. llama3.2をダウンロードする
3. リポジトリをクローンする
4. ライブラリをインストール
5.  アプリを起動する

## 使い方

1. ブラウザでアプリを開く
2. 質問を入力する
3. 検索ボタンを押す
4. AIが関連ドキュメントを参照して回答する
